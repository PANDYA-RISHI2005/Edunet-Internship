# Solar Energy Data Analysis

## Overview
This project analyzes a solar energy dataset related to sustainability using **Python, Pandas, and NumPy**. The dataset was cleaned by handling missing values.

## Features
- Data Cleaning
- Missing Value Handling

## Technologies Used
- Python
- Pandas
- NumPy
- Jupyter Notebook

## Dataset
The dataset contains information about:
- City and Country
- Solar Radiation (GHI)
- Annual Sunlight Hours
- Electricity Price
- Solar Installations
- Annual Energy Production
- ROI
- CO₂ Reduction
- Solar Viability Score
