# Edunet-Internship
#  Green Skills Using AI Technologies

This repository contains the work completed as part of my **Edunet Foundation Internship**, with a focus on developing Green Skills through the application of Artificial Intelligence (AI) technologies.

##  About the Internship

The Edunet Foundation Internship aims to equip learners with practical knowledge of emerging technologies while promoting sustainable development. This repository showcases the assignments, presentations, and coding exercises completed during the internship.

##  Repository Contents

* Presentation on **Green Skills Using AI Technologies**
* Covers the role of AI in promoting sustainability, environmental conservation, and resource optimization.

### 2. Python Basics Assignment (07-07-2025)

The Python assignment includes practical examples of:

* Set methods
